### Instructions - Step by Step
## 1- Collect the data:
  -The collected data are from this link below: 

https://cs2eo.org/search?searchMode=single-lat-long&productA=TEMPO_SWATH_POINT&minLat=-90&maxLat=90&minLong=-180&maxLong=180&startDate=2021-12-31&startTime=00%3A00&endDate=2022-01-31&endTime=00%3A00&trackSpacing=100&batchRequest=false

  -Parameter used was longitude and latitude for find every glacier point in the globe.
  -It's possible to download a csv file in the end of the search.
  -The csv file is on the path file `./src/input/cs2eo.csv`

## 2- Data formatting:
  -First the script read the csv data 
  -Then the script extract every porpeties of longitude and lagitude 
  -The script made a new list with the extract data to make future analysis easier 
  -The file create is on `./src/output/glacial-location.json`

## 3- Glacial image generation:
  -The data will be used on colab https://colab.research.google.com/drive/1ApK2ufFqPL_GR6_spfzIMRgK1nN7PZ4M?usp=sharing
  -This project have the goal to generate glacier images using latitude and longitude points
  -It's possible to set a date to the generation
  
## 4- Masks with AI:
  -Using the glacier image in colab https://colab.research.google.com/drive/1maW831SnWa-Lga-ExrviDKjg3BykI5EE?usp=sharing
  -An AI indetify the polar ice cap 
  -Been possible to understand the diference year by year
